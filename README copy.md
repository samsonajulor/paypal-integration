# Node.js PayPal REST SDK Sample

Example showing how to use the PayPal REST SDK with Node.js to take a payment

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser

```sh
$ npm start
```
